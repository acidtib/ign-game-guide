## IGN Game Guide - Discord Bot

### Add Bot To Your Server
https://discord.com/api/oauth2/authorize?client_id=1030253428573737041&permissions=0&scope=bot%20applications.commands

## Commands

### Search Wiki Guides
```
/ign-guides

options: 
	game: required
	keyword: optional
```

### Search Games
```
/ign-games

options: 
	game: required
```

### Search Articles
```
/ign-articles

options: 
	search: required
```

### Search Entire Site
```
/ign-site

options: 
	search: required
```
